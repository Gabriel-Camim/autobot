# Mercado

Minha visão de mercado é que IA, dados e automação só geram valor quando viram estrutura operacional. Não basta chamar API de LLM; é preciso saber onde o dado entra, como ele é validado, qual decisão apoia, qual risco reduz e como o sistema preserva contexto.

Eu gosto de problemas em que há muita informação dispersa, processo manual, decisão recorrente, baixa rastreabilidade ou dificuldade de transformar conhecimento em operação. É nesse tipo de cenário que RAG, workflows, bancos relacionais, agentes, dashboards e automações podem ter impacto real.
