# Pacote recrutador

O botão Extrair Gabriel baixa um pacote preparado para recrutadores. Ele não baixa toda a base de conhecimento; baixa apenas documentos curados e úteis para avaliação rápida.

Arquivos atuais do pacote:

- `Curriculo_Gabriel_Camim_Dados_IA_Automacao.docx`.
- `curriculo.md`.
- `perfil-tecnico.md`.
- `resumo-projetos.md`.
- `carta-motivacao.md`.
- `README.md`.

O pacote não inclui `.env`, chaves de API, índice Chroma, caches, logs, arquivos temporários ou qualquer material bruto fora do diretório `/materials/recruiter-pack`.
