# Currículo para recrutadores

O botão Extrair Gabriel baixa diretamente o currículo em DOCX de Gabriel Camim Santos. Ele não baixa toda a base de conhecimento e não gera mais um arquivo `.zip`.

Arquivo entregue pelo botão:

- `Curriculo_Gabriel_Camim_Dados_IA_Automacao.docx`.

O diretório de materiais ainda pode conter Markdown de apoio para a base de conhecimento, mas o download público direto entrega apenas o currículo em Word. Ele não inclui `.env`, chaves de API, índice Chroma, caches, logs ou arquivos temporários.
