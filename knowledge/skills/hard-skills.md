# Hard skills e Stack

## IA generativa, agentes especializados e LLMs

Orquestração via Langchain e N8N de modelos da openai como: 

- GPT Realtime 2 com WebRTC para agentes conversacionais de vóz em tempo real; 
- Text-embedding-3-large para conversão vetorial com armazenamento com pgVector em PostgreSQL; 
- Gpt-4o-mini-tts para conversão de texto em vóz; 
- Integração entre LLM's em processos de automação para rotina e ETL.
- Engenharia de prompts; 
- Agentes conversacionais multimodais e FAQ; 
- Structured outputs JSON; 
- Tool calling; 
- RAG; 
- Guardrails; 
- Ciclo automatizado para avaliação de respostas com feedback loop > Curadoria RAG > Seleção de documentos por proximidade semântica e distância vetorial > injeção de documentos externos convertidos em embeddings no banco vetorial relacionados por pontes semânticas ao contexto da curadoria > Prompt para sugestão de mudança na base de dados > Edição e avaliação de Diff gerado por IA > Alteração canônica versionada via GIT e reindexação RAG para aplicação canônica das mudanças no agente;

## Desenvolvimento e arquitetura (AI First)
 
Frontend com Vite + React com deploy em Vercel e Render para deploy de backend (JavaScript / Python), e database em postgresSQL. Metodologias ágeis para desenvolvimento, Rest API e Fast API, CRUD, versionamento Git e Cloud Computing. 

## Análise de Dados e Automação

- Domínio de N8N para processos ETL em data lakes e data warehouses; 
- Capacidade analítica e engenheira para orquestrar e integrar diferentes tecnologias para geerar insights de negócios a partir de dados extraídos;
- SQL, PostgreSQL, bancos vetoriais, modelagem relacional, schemas, KPIs, projeções, forecast, governança de dados, lineage, rastreabilidade, documentação de métricas, análise operacional, Power BI, Superset.
- Capacidade para identificar lacunas operacionais e transformar em esteiras automatizadas com foco em redução de custos, tempo de entrega e qualidade;
