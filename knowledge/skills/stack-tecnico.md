# Stack técnico

Stack técnica principal usada por Gabriel em projetos de dados, IA, automações, agentes conversacionais e sistemas AI First.

## IA generativa, agentes especializados e LLMs

- Orquestração de agentes, fluxos RAG e automações com LangChain e N8N.
- OpenAI API com uso dos modelos disponíveis conforme necessidade do caso de uso, sem fixar a stack em um único modelo de LLM.
- GPT Realtime 2 com WebRTC para agentes conversacionais de voz em tempo real.
- Text-embedding-3-large para conversão vetorial.
- PostgreSQL com pgVector para armazenamento relacional e vetorial.
- Gpt-4o-mini-tts para conversão de texto em voz.
- Engenharia de prompts.
- Agentes conversacionais multimodais e FAQ.
- Structured outputs em JSON.
- Tool calling.
- RAG.
- Guardrails.
- Ciclo automatizado de avaliação de respostas com feedback loop, curadoria RAG, seleção de documentos por proximidade semântica e distância vetorial, injeção de documentos externos convertidos em embeddings no banco vetorial, sugestão de mudança na base de dados, avaliação de diff gerado por IA, alteração canônica versionada via Git e reindexação RAG.
